# ピッグ・ピック・トリック（ピピとり）オンライン対戦プロトタイプ

部屋コードで4人が参加して遊べる、ブラウザ版オンライン対戦プロトタイプです。

## できること

- 部屋コード作成 / 参加
- 4人オンライン対戦
- 全員13枚の均一配札（ジョーカー以外の通常カードを1枚除外）
- マストフォロー
- ババブタ（ジョーカー）は場に出せない
- 勝者が場の4枚をごちそう山へ回収
- 勝者が最弱プレイヤーから1枚ピック
- ピック前に確認待ち時間あり
- おそろいペア浄化
- 2ラウンド制
- 最終スコア計算

## ローカルで動かす

```bash
npm install
npm start
```

その後、ブラウザで以下を開きます。

```text
http://localhost:3000
```

同じWi-Fi内のスマホで遊ぶ場合は、PCのローカルIPアドレスを使って開きます。
例：

```text
http://192.168.0.10:3000
```

## インターネット越しに友達と遊ぶには

このプロトタイプは WebSocket サーバーが必要です。
Render / Railway / Fly.io / 自分のVPSなどにアップすると、iPhoneからURLを開いてオンライン対戦できます。

## 注意

- まだプロトタイプなので、途中切断時の再接続・観戦・CPU補充は簡易仕様です。
- ルームはサーバーのメモリ上に保持されます。サーバーを再起動すると消えます。
- 本番公開する場合は、HTTPS対応のホスティングに置くのがおすすめです。
