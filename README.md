# ピピとりオンライン（CPUコメント＆演出版）

Render / Railway / Fly.io などに置いて遊べる、ピッグ・ピック・トリック（通称：ピピとり）のオンライン対戦プロトタイプです。

## 今回の追加内容

- CPUが状況に応じてコメントします。
  - 例：「うわっ、きついな〜」「まさか、ここで勝ちに行くブヒ！」「そこにババブタいないでほしいブヒ…」など
- 画面上部に「CPU実況」欄を追加しました。
- プレイヤー欄にCPUの吹き出しコメントを表示します。
- 直前トリックの「勝者」「最弱」を大きく表示します。
- 勝者枠・最弱枠を場のカード上でハイライトします。
- 出せる手札が浮くようにして、操作可能カードが分かりやすくなりました。
- ピック中の裏向きカードに揺れ演出を追加しました。
- ピック結果カードを大きく表示します。

## Renderへの反映

既存の GitHub リポジトリの中身を、このZIPの中身で上書きして Commit してください。
Render の Auto Deploy が有効なら自動で更新されます。

## ローカル起動

```bash
npm install
npm start
```

ブラウザで以下を開きます。

```text
http://localhost:3000
```

## 補足

今の段階では、CPUコメントは簡易AIです。今後、CPUごとに性格を付けたり、危険カードの保持状況や点数状況に応じてさらに濃いセリフを出すこともできます。
